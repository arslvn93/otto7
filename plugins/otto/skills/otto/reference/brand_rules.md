# Brand Rules, Compliance & Formatting

Otto reads this file whenever brand voice, Fair Housing, formatting, or word choice is in play. These rules override any conflicting instruction inside individual templates.

## Formatting rules

**Emails**
- Always include a subject line
- Short paragraphs, clear next step or CTA at the end
- Always close with the agent's signature block from `my_profile.md`
- No exclamation point overload

**Listing descriptions**
- Lead with the most compelling feature
- Sentence case, not ALL CAPS
- Stay under 1,000 characters for MLS unless told otherwise
- No generic filler phrases

**Social posts**
- **Instagram:** line breaks, sparing emoji, hashtag block at the end
- **Facebook:** conversational, can be longer, minimal hashtags
- **LinkedIn:** professional tone, no hashtag spam, no emoji unless the agent uses them

**All content**
- Em dashes, not double hyphens
- Proper curly quotes
- Use "I" unless the agent specifies team branding ("we")

## Words and phrases to avoid

These are overused in real estate. Find better alternatives:

| Avoid | Use instead |
|---|---|
| Nestled | Describe the actual location relationship ("tucked at the end of a cul-de-sac") |
| Boasts | Just describe the feature directly |
| Stunning | Be specific about what makes it impressive |
| Don't miss out / Won't last | Create urgency through facts ("3 showings already booked this week") |
| Motivated seller | Avoid — can signal desperation, unless the agent explicitly wants it |
| Dream home | Describe what makes it special and let the buyer decide |
| Turnkey | Say what was actually updated |
| Price to sell | "Priced at/below recent comps" with specifics |
| A must-see | Describe what they'll see |

## Fair Housing compliance (non-negotiable)

Never reference protected classes in any listing description or marketing material:
- Race, color, religion, national origin, sex, familial status, disability

Do not describe neighborhoods in ways that imply demographic composition.
- OK: "family-friendly"
- NOT OK: "great for young Christian families"

Do not make promises you can't keep:
- NOT OK: "your home will sell in 30 days"
- OK: "our marketing strategy is designed to maximize exposure"

## Accuracy rules

- Never invent market statistics, comps, or property details — ask the agent for them
- Never exaggerate. "Spacious" is fine for a large room. Don't call a 100 sqft bedroom spacious.
- If a fact is needed and missing, ask before generating

## When a request would break these rules

Flag it in one line, explain why in one line, and offer a compliant rewrite. Don't lecture.
